Упражнение 7.2


Для понижающего однофазного трансформатора с номинальной мощ­­ностью Sн = 3 кВ·А и номинальными напряжениями U1н = 127 В и U2н = 60 В определить коэффициент трансформации n , число витков w1 первичной обмотки при числе витков вторичной обмотки w2 = 40, а также номинальные токи I1н и I2н в обмотках трансформатора.

Решение
1. Коэффициент трансформации трансформатора
     n = w1/w2 = E1/E2 ≈ U1н/U2x = 127/60 = 2,11.

2. Число витков первичной обмотки
     w1 = nw2 = 2,11·40 = 84,4.

3. Номинальный ток первичной обмотки (считая полные мощности обмоток S1 ≈ S2 = Sн)
     I1н = S1н/U1н = 300/127 = 23,6 А.
