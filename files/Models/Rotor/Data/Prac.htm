Упражнение №1

При каком способе возбуждения ДПТ с увеличением тока якоря и постоянном токе возбуждения магнитный поток уменьшается?

 

Упражнение №2

ДПТ независимого возбуждения имеет следующие номинальные данные P2ном = 3кВт; Uном=220В; Iя.ном  = 15А; КПД=80%.Считая, что электрические потери в обмотке якоря в номинальном режиме составляют половину всех потерь, определить сопротивление обмотки якоря.

 

Решение:

Р1=Р2ном / h= 3*103 / 0.8 =3750 Вт.

Потери:

S DP =P1 - Р2ном = 3750 – 3000 =750 Вт.

Электрические потери:

DРэ.я = S DР / 2 = 750 / 2 = 375 Вт.

Сопротивление обмотки якоря:

Rя = DРэ.я / I2я.ном = 375 / 152 =1.66 Ом
