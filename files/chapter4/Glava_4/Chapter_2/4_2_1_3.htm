4.2.1.3. Решение однородного и полного уравнений

 

Решение однородного уравнения t [diLсв/dt]+ iLсв= 0 имеет вид
     iLсв = Aept ,
где A - постоянная интегрирования; p - корень характеристического уравнения tp + 1 = 0, равный p = -1/t = -a, где a = 1/t = R/L [1/с] - коэффициент затухания переходного процесса.
Полное решение уравнения (4.3):
     iL = I0+ Aept = I0 + Ae-at = I0 + Ae-t/t .

Постоянную интегрирования A определяют по начальным условиям. Положим, что при t = 0- ток iL(0-) = 0. Учитывая, что iL(0+) = iL(0-) = 0, из последнего уравнения (при t = 0+) iL(0+) = I0 + Ae-a·0 = I0 + A = 0 находим
     A = - I0.
Следовательно, переходный ток в индуктивном элементе L
     iL(t) = I0 - I0e-at = I0(1- e-at) = I0(1- e-t/t).
