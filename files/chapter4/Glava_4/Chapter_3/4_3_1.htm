4.3.1. Общие положения
Расчёт переходных процессов методом переменных состояния в электрических цепях предполагает:

  •формирование системы дифференциальных уравнений первого порядка в форме Коши (в нормальной форме) относительно переменных состояния цепи: токов iLk в индуктивных катушках и напряжений uCk на конденсаторах. Значения этих переменных должны быть известны к началу процесса;
•аппроксимацию уравнений состояния разностными уравнениями;
•численное решение полученной разностной системы уравнений на каждом шаге расчёта по времени.
     
В общем случае линейных и нелинейных цепей уравнения состояния формируют, исходя из топологических соотношений с построением графа цепи, в которой каждый элемент выделен в качестве ветви.
Для цепей с небольшим числом реактивных элементов система уравнений состояния может быть составлена с помощью уравнений Кирхгофа для послекоммутационной схемы, других методов, в т. ч. с помощью метода наложения решений для активной резистивной схемы замещения цепи, полученной в результате замены реактивных элементов источниками тока или напряжения.
