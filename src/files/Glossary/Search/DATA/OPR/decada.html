Декадa:
Декада - единица частоты при построении логарифмических частотных характеристик электрических величин. При логарифмировании по оси частот низкие частоты хорошо представляются на оси абсцисс. При этом, если отношение w2 / w1 = 10, то диапазон частот Δw называют декадой.



(Англ.): Decade
(Нем.):  Dekade f
(Фр.):   Décade /f/
