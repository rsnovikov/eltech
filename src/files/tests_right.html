Тесты


1.

Для проведения тестирования необходимо инсталлировать с компакт-диска на Ваш компьютер тестовую систему UTIS. Инсталляция пройдет автоматически после щелка мышью на выделенном слове УСТАНОВИТЬ.
2.Создать свою базу тестов можно в редакторе тестов UTISB, описание которого приведено в соответствующем файле дерева.
3.

Тестируемому по математике, физике, электротехнике (на первом уровне) по темам дисциплины и по группам тем последовательно выводятся на экран дисплея 15...20 тестовых вопросов, которые выбираются случайным образом из баз, в которых соответственно 80, 80, 30-40 и 300 вопросов. При желании можно установить (воспользовавшись программой UTISB) другие параметры тестирования (число заданий, время, вид оценки результатов тестирования и др.).
4.

В качестве примера организации письменного итогового тестировании по дисциплине "Основы теории цепей и сигналов" на втором уровне, проводимого в течение ряда лет в "МАТИ"-РГТУ им. К. Э. Циолковского по окончании студентами 3-го курса, приведена система выбора (случайным образом) двух заданий из базы, содержащей 100 заданий, и одного задания из базы, содержащей 16 заданий, подготовленных для предъявления студентам 5-го курса, при проведении аккредитации (лицензировании) кафедры.
