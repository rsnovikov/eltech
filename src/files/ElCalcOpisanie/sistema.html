Интерфейс подпрограммы "Системы уравнений"

Вызов подпрограммы "Системы уравнений" осуществляется щелчком мышью на одноимённой кнопке панели подпрограмм "Выберите режим работы калькулятора" (рис. 7).

Требуется, например, решить систему уравнений:

     [5+j4] Х1 - 8e(-60j) Х2 + j3 Х3 = -5;
     -8e(-60j) Х1 + 4e(30j) X2 - j4 Х3 = 0;
     j3 Х1 - j4 X2 + [3-j4] X3 = 8+j5.

Выбираем число переменных, равное 3, и вводим комплексные коэффициенты в соответствующие поля (рис. 8). После ввода свободных членов уравнений в поля после знака равно ("=") щёлкнем мышью на кнопке "Рассчитать!!!". Вычисленные программой определитель системы и значения переменных Х1...Х3 выводятся (в алгебраической и показательной формах) в соответствующих полях.

Если определитель равен нулю, выводится сообщение: "Определитель равен нулю, вычисления прекращены".

Имеется возможность сохранения исходных данных и результатов расчета системы уравнений. Для этого необходимо нажать кнопку "Сохранить" и указать имя файла. Чтобы открыть файл с сохраненными данными, нажмите на кнопку "Открыть" и выберите нужный файл. (Файл можно редактировать вручную, т. е. изменять порядок системы уравнений и значения ее коэффициентов).
