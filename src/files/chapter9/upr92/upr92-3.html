Упражнение 9.2
Двигатель постоянного тока параллельного возбуждения имеет следующие номинальные данные: мощность
Pн = 11 кВт; напряжение сети Uн = 220 В; частота вращения якоря nн = 1500 об/мин; номинальный ток Iн = 58 А; сопротивления обмоток якоря и возбуждения: Rя = 0,14 Ом; Rв = 80 Ом.
Определить номинальный ток и ЭДС якоря, номинальный вращающий момент двигателя, электромагнитную мощность, пусковой ток якоря при пуске без пускового реостата и сопротивление пускового реостата для обеспечения пускового тока I'яn = 2Iян.

Решение
1. Номинальный ток цепи возбуждения
     Iвн = Uн/Rв = 220/80 = 2,75 A.
2. Номинальный ток цепи якоря
     Iян = Iн - Iвн = 58 - 2,75 = 55,25 А.
3. ЭДС якоря
     Eян = Uн - RяIян = 220 - 0,14 · 55,25 = 212,26 В.
