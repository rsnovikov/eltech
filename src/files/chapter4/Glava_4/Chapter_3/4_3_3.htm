4.3.3. Формирование уравнений состояния методом наложения

 

Заменив индуктивные и ёмкостные элементы в послекоммутационной схеме соответственно источниками тока и напряжения, получают резистивную схему с дополнительными источниками. Записав напряжение uLk = L[diLk/dt] и ток iCk = C[duCk/dt] для каждого индуктивного и ёмкостного элемента, составляют систему уравнений состояния для n схем, в каждой из которых оставлен один источник. Полученную методом наложения систему уравнений состояния цепи решают, как правило, посредством численных методов на ЭВМ.

Для конкретизации этапов формирования уравнений состояния методом наложения рассмотрим еще раз послекоммутационную схему (см. рис. 4.36), выделив в ней ёмкостный С и индуктивный L элементы (рис. 4.38, а) так, чтобы оставшаяся часть схемы содержала только резистивные элементы R1 и R2 и источник напряжения е.

Согласно принципу компенсации индуктивный элемент L заменим источником тока JL = iL, который доставляет ток в том же направлении, что и в исходной схеме, с напряжением на нём uL = LdiL/dt, а ёмкостный элемент С - источником напряжения с ЭДС EC = uC, направленной встречно току iС (рис. 4.38, б).
