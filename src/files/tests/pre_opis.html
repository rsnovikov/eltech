Описание претестов


Претесты - это тесты для оценки усвоения знаний разделов физики и высшей математики, на которых базируются электротехнические дисциплины. Предварительный контроль знаний необходим для выработки стратегии обучения с возможным разделением студентов, по меньшей мере, на два потока. Если же подготовленность обучаемого ниже определенного уровня, то вряд ли удастся достичь удовлетворительных результатов в отведенные часы на изучение электротехнической дисциплины.
Примеры претестов:
1. Тест на опознание: "Укажите (Да/Нет?), выражение A = 5ej1370 равно выражению A = -3 + j4?.

Эталон (1 с. о., где с. о. - существенная операция): Да.

2. Тест на различение: "Укажите, чему равен ln10:
  а) 1;   б) 2;   в) ~2.3;   г) 4.6.
Эталон (1 с. о.): в.

3. Тест на классификацию: "Укажите, какие признаки из приведенных:
  а) механическое воздействие на заряженное тело;
  б) основные величины ;
  в) основные величины ;
  г) появление ЭДС в движущемся проводнике;
  д) ,
характеризуют:
  1. электрическое поле;
  2. магнитное поле".
Эталон (5 с. о.): 1. б, д.  2. а, в, г.
