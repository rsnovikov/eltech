Описание аккредитационных тестов


Аккредитационные тесты - тестовые задания по электротехнической дисциплине, в которые входят тесты по всем дисциплинам научного направления для оценки конечного результата подготовки специалистов. Это тесты 2-го и З-го уровней, большинство из которых профессиональные стыковые. Аккредитационное тестирование проводится перед государственными экзаменами после окончания цикла обучения для аккредитации кафедры и вуза.
