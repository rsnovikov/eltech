Задание 3. Собрать на рабочем поле программы EWB или рабочем поле программы MS8 Education схему цепи (рис. 14.5) (или открыть файл LR14.ewb при выполнении работы в среде EWB, или файл Circuits\Lr14-1.ms8 при выполнении работы в среде MS8SD), подключив выход функционального генератора FG и входы осциллографа к указанным на схеме узлам. Управляемый напряжением источник тока ITUN включен в схему для снятия напряжения uR(t) с резистора R, пропорционального току i(t). Для получения соответствующей напряжению uR(t) масштабной сетки на экране осциллографа коэффициент передачи (Ом) источника ITUN должен быть равным сопротивлению R.


Установить параметры пассивных элементов схемы, рассчитанных в задании 1. При этом установить (см. рис. 14.5, в правом верхнем углу, например, для варианта N = 31): сопротивление потенциометра R = 2Rкр; его уровень Setting = 50%; шаг изменения Inckrement = 5%; выбрать управляющей клавишу (Key) R. Для уменьшения сопротивления R потенциометра на 5% необходимо вначале его выделить (щёлкнув мышью на изображении элемента R), а затем нажать клавишу R. Для увеличения сопротивления на 5% необходимо одновременно нажать клавиши Shift и R.

Установить параметры функционального генератора FG (напряжение, частоту, смещение (Offset)) и осциллографа, ориентировочные значения которых приведены на рис. 14.5. При этом длительность импульса генератора рекомендуется установить такой, чтобы выполнялось условие   tи > (5-8)τ.

Положение управляющих ключей:

- при исследовании RL-цепи ключ Q замкнут (управление - посредством нажатия на клавишу Q клавиатуры), а ключ S разомкнут (управление -посредством нажатия на клавишу S клавиатуры);

- при исследовании RС-цепи ключ S замкнут, a ключ Q разомкнут;

- при исследовании RLС-цепи оба ключа разомкнуты.

Внимание! При снятии осциллограмм uL(t) и uС(t) схему цепи необходимо скорректировать таким образом, чтобы элемент цепи, с которого снимается осциллограмма, имел один заземлённый полюс (вывод). Необходимость этого условия вызвана тем, что один из входных зажимов осциллографа постоянно заземлен.

Примечание. Исследование переходных процессов в цепях проводить на расширенной модели осциллографа (щёлкните мышью на клавише Espand) и при положениях переключателей горизонтальной развёртки луча, обеспечивающих развертку длительности переходного процесса на 0,6...0,8 ширины экрана осциллографа.

Скорректировать опции команд программы EWB (MS8 Education). Например, удовлетворительные графики переходных величин получены при следующих опциях команд:

 - Analisis\Analysis Options\Transient\Transient Tolerance Factor = 0,1;

- Analysis Options/Instrument\Pause after each screen – установить;

                                                \Points per cycle  =  400;

                                                \Generate time steps… = 500.
