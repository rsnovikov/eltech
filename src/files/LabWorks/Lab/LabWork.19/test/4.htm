4. Укажите, с какой целью на щиток двигателя выводятся начала и концы (всего 6) трех обмоток статора?


Для подключения обмоток статора к трехфазной сети переменного тока.

Для выбора схемы соединения обмоток статора: звездой или треугольником.

Для замены местами двух проводов из трех, подводящих трехфазное напряжение сети к началам обмоток статора для реверсирования АД.

Для изменения числа пар полюсов магнитного поля статора и уменьшения синхронной скорости АД.
