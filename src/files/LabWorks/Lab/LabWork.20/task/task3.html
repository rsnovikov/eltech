Задание 3. По данным моделирования процессов в МПТ рассчитать ток I и мощность P1, потребляемые двигателем из сети, выходную мощность P2 на валу и КПД η двигателя при разных нагрузках, воспользовавшись следующими расчётными формулами:

I = Iя  + Iв, где Iя - ток в обмотке якоря двигателя, А;

Р1 = UнI, где Р1 - электрическая мощность, потребляемая двигателем  из сети, Вт (выразить её в кВт);      

P2 = M·n / 9550,

где Р2 - полезная механическая мощность на валу двигателя, кВт;  М - момент вращения  якоря двигателя, Нм; n - частота вращения якоря, об/мин;  η = Р2/ Р1 - коэффициент полезного действия ДПТ.

Результаты расчётов I, P1, P2 и η занести в табл. 20.1.

По результатам вычислительного эксперимента и расчётов (см. табл. 20.1) построить механическую n = f(M) (см. рис. 20.4) и рабочие характеристики n = f(Iя), M = f(Iя) и η = f(Iя) ДПТ (совместив их в одних координатных осях, см. рис. 20.5) в относительных единицах, приняв за условные единицы номинальную частоту вращения вала nн, номинальный момент вращения Мн и номинальный ток в обмотке якоря Iян.
